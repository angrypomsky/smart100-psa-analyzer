# 2. 분석 대상 및 판정 기준

## 2.1 대상 노형 및 사고유형 분류

- SMART 일체형 원자로 개요 (PRHRS 4계통 구성)
- 5개 사고유형 분류 체계: LOFW, SBLOCA, GTRN, LSSB, SGTR
- 각 사고유형의 물리적 특성 간략 설명 (표 1개로 정리)

## 2.2 안전 기능 판정 기준

- CD 판정: NRC 10 CFR 50.46 기준 PCT 1477 K
- PRHRS 성공 기준: IAEA SSG-3 기반 (4/4, 3/4, 2/4 등)
- RT 성공 판정 기준: rktpow 80% 감소

## 2.3 열수력 해석 코드 및 입출력 구조

- MARS-KS 코드 개요
- 시뮬레이션 출력 파일 구조 (Excel, 시간-변수 행렬)
- 변수명 파일의 구조 (vartype, var_id, description)
- 출력 변수 종류: rktpow, PRHRS HX 열출력(4개), PCT(피복재 온도), RCP 헤드

# 3. 알고리즘 설계

## 3.1 변수명 자동 매핑 (VarMapper)

- 문제 정의: MARS-KS 출력 컬럼명이 vartype 기반 → 물리적 의미를 알 수 없음
- 해결: 변수명 파일의 description 컬럼에서 키워드 매칭으로 역할별 자동 분류
- 매핑 역할 3가지: PRHRS_HX, PCT, RCP_HEAD
- 키워드 매칭 규칙 표
- 노형 확장성: 키워드 딕셔너리만 수정하면 다른 노형 적용 가능

## 3.2 원자로 트립(RT) 감지 알고리즘

- RT 감지 원리: rktpow가 초기값 대비 80% 감소한 첫 시점
- RT 감지와 PCT용 RT의 구분 (80% vs 90% 감소 기준)
- Fallback 처리: 데이터가 RT 이후부터 시작하는 경우 (GTRN, LSSB 등)
- 알고리즘 흐름도 (그림)

## 3.3 PRHRS 계통수 자동 판정 알고리즘 (핵심)

- 기존 절대 임계값 방식의 한계 설명
- 상대 비교 알고리즘 제안: val >= max_val x RATIO
- 알고리즘 5단계 흐름:
  1. RT 이후 WAIT 대기
  2. 4개 HX 열출력 집계 (집계방식 선택)
  3. FLOOR 미만이면 0계통 판정
  4. max_val x RATIO 이상인 계통 수 산출
  5. CORRECTION 보정 적용
- 알고리즘 흐름도 (그림)
- 사고유형별 파라미터 설정 및 물리적 근거 (표):
  - 집계방식 (전체평균 vs 후반30%)
  - WAIT, FLOOR, RATIO, CORRECTION
- LSSB 후반 30% 평균 선택 근거 (초기 과도현상 배제)
- SGTR CORRECTION=-1 적용 근거 (파단 유로 HX 보정)
- 수식 정리

## 3.4 PCT 최대값 추출 알고리즘

- RT(90% 감소) 이후 구간 필터링 근거
- 이상값 제거 기준 (> 10,000 K)
- cladding temp 키워드 기반 자동 컬럼 추출 (VarMapper 연동)
- PCT 최대값 및 발생 시점 추출 로직

## 3.5 RCP 상태 판정 알고리즘

- RT 이후 200초 구간의 RCP 헤드 평균값 기반 분류
- 3가지 상태: Running / Coast-down / Natural Circulation
- 판정 기준값 표

## 3.6 객체지향 설계 구조

- BaseAnalyzer: 공통 로직 (RT 감지, PRHRS 판정, PCT 추출)
- 사고유형별 서브클래스: 파라미터만 오버라이드
- 클래스 상속 구조도 (그림)
- 사고유형별 오버라이드 파라미터 비교 표

# 4. 적용 결과 및 검증

## 4.1 검증 데이터셋

- 사고유형별 시나리오 수, 데이터 출처
- 데이터셋의 구성 및 특성 (시뮬레이션 조건 범위)

## 4.2 RT 감지 결과 검증

- 사고유형별 RT 감지 결과 vs 수작업 확인 결과 비교 표
- 일치율 및 불일치 사례 분석

## 4.3 PRHRS 계통수 판정 결과 검증

- 사고유형별 판정 결과 vs 수작업 확인 결과 비교 표
- 일치율 및 불일치 사례 분석
- PRHRS 계통수 분포 (사고유형별)

## 4.4 PCT 추출 결과

- 사고유형별 PCT 분포 통계 (평균, 최대, 95th percentile)
- CD 판정 결과 요약

## 4.5 CD 임계값 민감도 분석

- 1300~1600 K 범위에서 CD 건수 변화
- NRC 기준(1477 K) 대비 결과 비교 표/그림
- 민감도 분석이 알고리즘 적용 가능성을 어떻게 뒷받침하는지

## 4.6 한계사항 및 적용 범위

- 검증 데이터셋이 OK 시나리오 위주로 구성된 한계
- ADS/PSIS/SIT 능동계통: 현재 데이터셋에서 작동 임계값 미달 → N/A 처리
- CD 시나리오에 대한 직접 검증은 향후 과제
